This archive contains additional data that you may need
for launch Xash3D.

1. valve/gfx/shell - WON-menu pictures set (if you have steam version without this images)
2. valve/cl_dlls - not crypted client.dll with some fixed bugs (optional)
3. valve/dlls - hl.dll with some fixed bugs (optional)
4. full source code of engine, menu.dll and original half-life SDK 2.3 code
Also new build contains makefont.exe which supports extended charset.
6. docs. all available documentations for engine.
